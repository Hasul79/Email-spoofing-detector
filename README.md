# Email-spoofing-detector

<br/>

<h2>Перейдите на свою электронную почту и выберите сообщение, которое хотите отсканировать. Вам нужно увидеть исходное сообщение целиком, вам нужно нажать на трехточечное меню («Дополнительные параметры») в правом нижнем углу окна электронной почты и выбрать «Показать оригинал». Откроется новое окно или вкладка с необработанным содержимым электронного письма, включая заголовки и тело. Нажмите «Загрузить оригинал» и загрузите файл .eml.</h2>
